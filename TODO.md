# 后续优化 TODO

## 第一优先级

- [ ] 逐题参考故事稿
  - 当前看图讲故事是通用参考框架，不是按图片内容逐题编写。
  - 为 175 组图片逐题生成 60 秒以内英文参考故事。
  - 每题建议提供两个版本：简单版、提高版。

- [ ] 语音自动转文字
  - 当前支持录音/录像，但文字稿需要手动填写。
  - 后续接入语音识别，把孩子讲的内容自动转成文字。
  - 家长可以二次编辑识别结果。

- [ ] 练习记录页优化
  - 每道题显示练过几次、最近一次日期、最高分、最近一次文字稿。
  - 支持筛选：未练、已练、薄弱题、收藏题。
  - 支持按环节查看练习历史。

- [ ] GitHub Pages 线上可用性检查
  - 检查录音/录像在 HTTPS 的 GitHub Pages 下是否稳定。
  - 明确提示录音录像保存在当前浏览器本地，换设备不会自动同步。

## 第二优先级

- [ ] 孩子友好的鼓励反馈
  - 练完后显示一句具体鼓励，而不是只显示分数。
  - 示例：今天故事有清楚的结尾；下次试试声音更响亮。

- [ ] 主题演讲专项模块
  - 支持上传/展示 A3 图片。
  - 记录 60 秒演讲稿版本。
  - 拆成开头、城市亮点、结尾三个小训练。
  - 支持只练开头 10 秒和完整 60 秒彩排。

- [ ] 观点论述素材库
  - 每个观点题提供 Agree 理由、Disagree 理由、生活例子、常用句型。
  - 帮孩子从背模板过渡到能说自己的理由。

- [ ] 总决赛高压问答题库
  - 根据电梯演讲主题自动给 3-5 个追问。
  - 记录孩子回答是否切题、是否有例子、是否能补救卡顿。

## 第三优先级

- [ ] 备赛日历
  - 从现在到 8 月中旬分阶段安排。
  - 首页显示本周建议，但不强制打卡。
  - 家长可以看到最近练习是否均衡。

- [ ] 成长曲线
  - 按环节显示完成次数、平均分、超时次数、常见问题。
  - 重点展示孩子是否越来越稳定。

- [ ] 错题/难题复练
  - 家长可以把某题标为重点复练。
  - 首页优先推荐重点复练题。
  - 已经练熟的题可以暂时隐藏。

- [ ] 导入/导出备份
  - 导出完整练习记录。
  - 支持导入备份，避免换浏览器后记录丢失。

## 建议下一步

1. 逐题生成看图讲故事参考稿。
2. 给观点论述题补 Agree / Disagree 参考理由。
3. 优化练习记录页，让家长快速看到哪些题练过、孩子当时怎么讲。
