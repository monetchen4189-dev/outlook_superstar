# 希望之星少儿A组备考训练

本地网页应用，用于 2026 希望之星少儿A组备考。

## 启动

在本目录运行：

```bash
python3 -m http.server 5173
```

然后打开：

```text
http://localhost:5173
```

录音和录像需要通过 `localhost` 打开，直接双击 `index.html` 时浏览器可能会禁止摄像头、麦克风或题库数据读取。

## 已包含

- 第一环节：主题演讲训练页
- 第二环节：175 组看图讲故事题库
- 第三环节：55 个观点论述题
- 总决赛：电梯演讲和高压问答模拟
- 首页四个环节入口，点击后直接进入练习
- 计时器、练习记录、文字稿、评分、备注
- 浏览器本地录音/录像保存与回放

## 数据位置

练习记录保存在当前浏览器本地，不会上传到网络。更换浏览器或清除浏览器数据后，记录会消失。可以在“练习记录”页导出 JSON 备份。
